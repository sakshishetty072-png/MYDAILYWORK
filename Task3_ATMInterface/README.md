# ATM INTERFACE

## How to Run
javac ATMInterface.java  
java ATMInterface

## Features
- Simple console-based ATM system
- User can deposit money
- User can withdraw money
- Check current account balance
- Validates invalid deposit and withdrawal amounts
- Prevents withdrawal if insufficient balance
- Menu-driven interface using switch-case
- Uses Object-Oriented Programming (OOP) concepts
- Separate classes for ATM and Bank Account
